<h4>badge basic use</h4>

```js
    <Badge>simple badge</Badge>
    <Badge solid>simple badge</Badge>
    <Badge>1</Badge>
    <Badge solid>100</Badge>
```

<h4>badge wrapper use</h4>

```js
<Badge.Wrapper content={100}>
  <h1 style={{ border: '1px solid #000', margin: 0 }}>Badge wrapper</h1>
</Badge.Wrapper>
```
