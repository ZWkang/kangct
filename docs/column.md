
<h2>simple use</h2>
<h5>default 12 columns</h5>
```js
<Column is-12>
    <Column xm-is-1 border>1</Column>
    <Column is-1 border>1</Column>
    <Column is-2 border>2</Column>
    <Column is-4 border>4</Column>
    <Column is-4 border>4</Column>
</Column>
```

```js
<Column is-12 border>
    <Column md-is-6 sm-is-4 xl-is-12 is-2 lg-is-9 gutter="1rem" border>1</Column>
</Column>
```

```js
<Column is-12-12>
    <Column is-3-4 border>3/4</Column>
    <Column is-1-12 border>1/12</Column>
    <Column is-1-12 border>1/12</Column>
    <Column is-1-24 border>1/24</Column>
    <Column is-1-24 border>1/24</Column>
</Column>
```

```js
<Column is-12-12>
    <Column is-6 left-offset-1 border>6</Column>
    <Column is-4 left-offset-1 border>4</Column>
    <Column is-11 right-offset-1 border>11</Column>
    <Column is-5 left-offset-7 border>11</Column>
    <Column is-3-4 left-offset-1 border>11</Column>
</Column>
```

```js
<Column is-12-12>
    <Column is-3-4 left-offset-1-4 border>3/4</Column>
    <Column is-1-2 left-offset-1-4 border>1-2</Column>
    <Column is-1-4 border>1-2</Column>
</Column>
```
